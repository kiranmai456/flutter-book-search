# 📚 Flutter Book Search App

A modern Flutter application built with **Flutter** and **Dart** that enables users to search for books and explore detailed information such as title, author, cover image, and description. The application consumes a **REST API** to fetch real-time book data and uses **Redux** for centralized state management, following a clean and scalable architecture.

---

## ✨ Features

* 📖 Search books by title or keyword
* 📚 View detailed book information
* 🖼️ Display book cover images
* 🌐 REST API integration
* 🔄 Redux state management
* ⚡ Loading & error handling
* 🧩 Reusable widgets
* 📱 Responsive and intuitive UI

## 📂 Project Structure

```text
lib/
│
├── data/
├── model/
├── pages/
├── redux/
├── utils/
├── widgets/
└── main.dart
```

### Folder Description

* **data/** – Handles API requests and data fetching.
* **model/** – Contains Dart model classes for JSON parsing.
* **pages/** – Application screens and user interface.
* **redux/** – Centralized application state management.
* **utils/** – Utility methods, constants, and helper functions.
* **widgets/** – Reusable UI components.
* **main.dart** – Entry point of the application.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-book-search.git
```

### Navigate to the project

```bash
cd flutter-book-search
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
